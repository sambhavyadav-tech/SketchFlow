
# FlowSketch Ultimate

AI-ready smart diagramming board.

## Included Features

✅ Infinite canvas
✅ Whiteboard drawing
✅ Modern toolbar
✅ Smart UI layout
✅ AI panel placeholder
✅ Undo/Redo buttons
✅ Dark mode
✅ Export placeholder
✅ Cloudflare deployment ready
✅ GitHub ready

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy To Cloudflare

Framework preset:
```
Vite
```

Build command:
```
npm run build
```

Output directory:
```
dist
```
